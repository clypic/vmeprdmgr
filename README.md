#VME.Co Front-End Developer Exercise
25th February 2017

This project is in development.

##Requirments
- docker v17 and up.
- docker-compose v1.11 and up.
- node v7.6.0

##Getting Started

1. Start the stack
```sh
$ docker-compose up -d
```
2. Open in Browser
- [Website](http://localhost:8080)
- [Backoffice](http:localhost:8081)
-
3. Shut down the stack
```sh
$ docker-compose down
```
